# 🛍️ RangoliStore - E-commerce Website  

RangoliStore is a modern e-commerce platform designed for seamless shopping experiences. Built with **React, Node.js, and Strapi**, it offers dynamic product management, secure authentication, and a responsive UI.  

## 🌟 Features  
- 🛒 **Product Management** – Powered by Strapi (Headless CMS)  
- 🔐 **Authentication** – Secure user login & registration  
- 🛍️ **Shopping Cart** – Add, remove, and purchase products  
- 📱 **Responsive UI** – Optimized for all devices  
- 🚀 **Scalable Backend** – Built with Node.js & Strapi  

## 🛠 Tech Stack  
- **Frontend:** React.js (with Tailwind CSS if used)  
- **Backend:** Node.js, Strapi  
- **Database:** PostgreSQL / MongoDB (whichever you're using)  
- **Hosting:** Deployed at [`rangolistore.me`](https://rangolistore.me)
